Jam Search NyMag Reader Pattern Catalog App

Pattern Catalog Database

This project is very close to my heart. It is two sided. The first side is an application for my business that allows me to add a catalog page to my amazon bucket and save it to a mongoDB database with other attributes. These database enteries will allow a user to search our rare catalog collection for their pattern with our user interface.

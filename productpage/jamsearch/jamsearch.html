Jam Search

Jam Search is a web based application that utlizes Bands in Town and Youtube APIs to deliever up to date content on bands and musical acts for the user. The user signs in using Google Firebase and searches for an artist. The API returns information on the artist and their upcoming shows. A box appears on the left if there are any upcoming events. The pill buttons allow the user to buy tickets on the venue's webpage. The youtube api returns the top 5 videos for that artist. On the right hand side we display an image of the artist along with their name.
